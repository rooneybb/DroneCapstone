Installing/Configuring Android Studio

1.       Please verify the JDK version is 1.8 or above.  By typing "javac -version" on your command prompt line.

1.1.  

 

2.       Verify or create environment variable “JAVA_HOME” is set to your JDK installed directory

2.1.  

 

3.       Review the system requirements for Android Studio/SDK at http://developer.android.com/sdk/index.html#Requirements

 

4.       Download and install Android Studio

4.1.   Go to https://developer.android.com/studio/index.html
